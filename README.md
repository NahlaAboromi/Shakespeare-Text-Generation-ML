# Shakespeare Text Generation using BERT

Machine learning project developed as part of the **Data Mining and Machine Learning Systems** course.

The project focuses on generating Shakespeare-style text using a Transformer-based model.

## Technologies
- Python
- BERT / BertGeneration
- HuggingFace Transformers
- NLP
- Jupyter Notebook
- Google Colab

## Description
The model was trained on large Shakespeare-style datasets to learn the structure and vocabulary of classical English literature.  
After training, the model can generate new text passages that imitate Shakespeare's writing style.

## Project Goals
- Explore Transformer-based language models
- Apply Natural Language Processing techniques
- Train and evaluate a text generation model
